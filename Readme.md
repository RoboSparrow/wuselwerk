# Wuselwerk

W.I.P. Sandbox simulation using ~~[SDL](https://www.libsdl.org/)~~ OpenGL/GLFW

![screenshot](./assets/screenshot.png)

---

## Getting Started

### Install

```bash
sudo apt-get install libglfw3-dev freeglut3-dev libglew-dev
```

### Build

```
make clean && make
./wusel
```

### Usage

```bash
./wusel -h
./wusel -c 300 -f 24
```
