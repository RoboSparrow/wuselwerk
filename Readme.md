# Wuselwerk

W.I.P. Sandbox simulation using [SDL](https://www.libsdl.org/)

---

## Getting Started

### Install

```bash
sudo apt install libsdl2-dev libsdl2-ttf-dev
```

### Build

```
make clean && make
./wusel
```

### Usage

```bash
./wusel -h
```
